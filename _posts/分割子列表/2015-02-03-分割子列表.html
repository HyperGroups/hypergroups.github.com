按条件分割子列表

{% raw %}

list = {4, 11, 15, 8, 1, 10, 5, 9, 7, 41, 2, 11, 41, 1, 2, 8, 2, 6, 1}
{% endraw %}

{% raw %}

f[list_] := Module[{p, acc, len}, p = list;

  acc = TakeWhile[Accumulate@p, # < 50 &];

  len = Length@acc; Sow[Take[p, len]]; Drop[p, len]]
{% endraw %}

{% raw %}

{rest, parts0} = Reap@FixedPointList[f, list]
{% endraw %}

{% raw %}

parts1 = parts0[[1]] // Most
{% endraw %}

f肯定不是一个好函数，每次都用了一下Accumulate[虽然这货很快]，临时为了进度写的，这是Mma的函数，脑子不好使的时候，可以先用把问题搞定，再优化代码。搞着搞着，我突然想到新版的神器，爽

{% raw %}

SequenceCases[list, list_ /; Total[list] < 50]
{% endraw %}

把列表按长度列表分割

{% raw %}

lengthList = Length /@ parts1
{% endraw %}

{% raw %}

parts = {#[[1]] + 1, #[[2]]} & /@ 

  Partition[Prepend[Accumulate@lengthList, 0], 2, 1]
{% endraw %}

{% raw %}

Take[list, #] & /@ parts
{% endraw %}

{% raw %}

FoldPairList[TakeDrop, list, lengthList]
