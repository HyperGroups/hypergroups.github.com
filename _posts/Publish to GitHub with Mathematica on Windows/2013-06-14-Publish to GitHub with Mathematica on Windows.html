Publish to GitHub with Mathematica on Windows

@2013-06-14

I’m on windows 8.1

Let’s just concentrate on how to commit the files. [git push]

Method

1 Auto type username and password

download an assistant tool小助手[Permanently authenticating with Git repositories]

my git-credential-winstore.exe is put and installed here

E:\\Users\\Hyper\\AppData\\Local\\GitHub\\PortableGit_fed20eba68b3e238e49a47cdfed0a45783d93651\\bin

2 Put commands in the .bat file

Put the commands set used in GitShell in the .bat file of the portable version of GitHub.

{% raw %}

batFile = 

  "E:\\Users\\Hyper\\AppData\\Local\\GitHub\\PortableGit_\

fed20eba68b3e238e49a47cdfed0a45783d93651\\git-cmd.bat";
{% endraw %}

{% raw %}

batSample = 

  "@rem Do not use \"echo off\" to not affect any child calls.

  @setlocal

  

  @rem Get the abolute path to the current directory, which is \

assumed to be the

  @rem Git installation root.

  @for /F \"delims=\" %%I in (\"%~dp0\") do @set git_install_root=%%~fI

  @set PATH=%git_install_root%\\bin;%git_install_root%\\mingw\\bin;%\

git_install_root%\\cmd;%PATH%

  

  @if not exist \"%HOME%\" @set HOME=%HOMEDRIVE%%HOMEPATH%

  @if not exist \"%HOME%\" @set HOME=%USERPROFILE%

  

  @set PLINK_PROTOCOL=ssh

  @if not defined TERM set TERM=msys

  

  @cd %HOME%

  ::@start %COMSPEC%

  

  ::@start \"\" cmd /k @cd /d documents\\github && @git

  

  cd /d documents\\github\\hypergroups.github.com

  ";
{% endraw %}

{% raw %}

commandsSet = "git add --all

  git commit -m \"" <> windowTitle[EvaluationNotebook[]] <> "\"

  git push"
{% endraw %}

{% raw %}

batFileToExport = batSample <> commandsSet(*<>"\npause"*);
{% endraw %}

{% raw %}

Export["bat.txt", batFileToExport];

DeleteFile@batFile;

CopyFile["bat.txt", batFile];
{% endraw %}

3 Run the .bat file

{% raw %}

batFile // SystemOpen
{% endraw %}
