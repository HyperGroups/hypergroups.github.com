JSON无能的一些情况

@2015-10-20

外部获得的一些结构化数据，导入时，若有些小问题，要加些预处理

python中字典套字典的数据

{% raw %}

str = "{'1':{'a':1,'b':1},'2':{'a':1,'b':1}}";
{% endraw %}

{% raw %}

ImportString[str, "JSON"]
{% endraw %}

在这个问题中，是因为从控制台复制过来的字符串的单引号的问题，有的时候，像这种小细节要尝试一些时间后才发现。
含有u’的unicode字符串则要把这个也替换掉再导入。

{% raw %}

ImportString[StringReplace[str, "'" -> "\""], "JSON"]
{% endraw %}
