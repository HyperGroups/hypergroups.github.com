MathKernel-LinkLaunch刷博客访问量的一个示例

@2013-05-01

多加载几个核心刷访问量玩一下。

{% raw %}

link = LinkCreate[LinkProtocol -> "TCPIP"]
{% endraw %}

{% raw %}

$CommandLine
{% endraw %}

{% raw %}

kernel = LinkLaunch[First[$CommandLine] <> " -mathlink"]
{% endraw %}

{% raw %}

LinkWrite[kernel, EnterTextPacket["10!"]]
{% endraw %}

{% raw %}

LinkRead[kernel]
{% endraw %}

{% raw %}

LinkClose[kernel]
{% endraw %}

{% raw %}

links = Table[LinkLaunch["math -mathlink"], {10}]
{% endraw %}

{% raw %}

 start = SessionTime[]

LinkWrite[#, 

     EnterExpressionPacket[

      Do[URLFetch[

        "http://quaternions.blog.163.com/blog/static/\

206082147201331414217418/"], {3}]]] & /@ links; // Timing

end = SessionTime[]

time = (end - start)
{% endraw %}

{% raw %}

start = SessionTime[];

times = Join[{0.},

     "http://quaternions.blog.163.com/blog/static/\

206082147201331414217418/"]; Sow[SessionTime[]], {30}] - start]
